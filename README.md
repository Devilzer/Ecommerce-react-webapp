## [`Ecommerce Webapp`](https://devilzer.github.io/Ecommerce-react-webapp/)

## Features

* In-Line product edit.
* Add new product.
* Alert notifications.
* Cart mangement.
* Persistant redux state.

## How to install and run?

1. Clone the repositories.
2. Install all dependencies by `npm install`
3. `npm start`
